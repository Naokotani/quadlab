## Quadlet Home Lab
Personal home lab setup using podman quadlets to manage services.

## What's inside?

- Caddy: For reverse proxy
- coredns: I use this to resolve domains running locally to the appropriate private IPv4 Address
- ddns: This is a simple go script I wrote to up date my IP address if my ISP changes it on me
- Jellyfin: Media server for music, movies etc.
- FoudnryVTT: This is a ttrpg virutal table top web application
